Device-family Note

The validated contiguous subset on this target is:

  • STS10, STC10, STN10, TS10, TC10, TN10
  • CS10, CC10, CN10
  • SB10, SW10
  • X10, Y10
  • M100, L100, F100
  • B10, D100, W10, Z10, R100

Keep DX, DY, V, and ZR out of the FX5U serial 3C/4C validated subset.

The FX5 serial communication manual's List of Commands and Functions (3C/4C frame, printed page 718-719) lists 0401/1401, 0403/1402, 0406/1406, remote control, loopback, and 1617, but does not list 0613, 1613, 0601, or 1601.
