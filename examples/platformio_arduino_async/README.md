# PlatformIO Arduino Async Example

This directory contains the shared Arduino-style asynchronous example with a simulated response path.

Read this after the simpler host-side example:

- [../host_sync_quickstart.cpp](../host_sync_quickstart.cpp)

Main file:

- [platformio_arduino_async.cpp](platformio_arduino_async.cpp)

Use this example when you want to understand the low-level async control flow without touching a real PLC line.

If you want a read-only real UART sample, use:

- [../platformio_arduino_uart/README.md](../platformio_arduino_uart/README.md)
