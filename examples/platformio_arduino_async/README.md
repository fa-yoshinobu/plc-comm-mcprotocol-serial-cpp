# PlatformIO Arduino Example

This directory contains the shared Arduino-style PlatformIO example with a simulated response path.

Main file:

- [platformio_arduino_async.cpp](platformio_arduino_async.cpp)

Use this example when you want to understand the control flow without touching a real PLC line.

If you want a read-only real UART sample, use:

- [../platformio_arduino_uart/README.md](../platformio_arduino_uart/README.md)
