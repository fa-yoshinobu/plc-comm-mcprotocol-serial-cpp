# RJ71C24-R2 RS-232C Report

Current-only summary for the `RJ71C24-R2` serial path.

- Serial module: Mitsubishi `RJ71C24-R2`
- Validated PLC CPUs:

## Current Rules

| Area | Rule |
|---|---|
| General contiguous/native family traffic | use `--series ql` |
| `SM`, `SD`, `RD`, `LZ`, `Jn\...`, long current-value spot checks | use `--series iqr` |

| `LTS/LTC/LSTS/LSTC` | interpret from structured `LTN/LSTN` `0401` responses, not as standalone direct-read targets; on this target the contact/coil bits were observed in the third response word, not the second |
